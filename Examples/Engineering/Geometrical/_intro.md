[CalcpadCE](https://calcpad-ce.org) bundles a worksheet for every cross-section that turns up in [structural analysis](https://en.wikipedia.org/wiki/Structural_analysis) — from primitive shapes to the full catalogue of rolled and built-up steel profiles.

Each page returns the same set of section properties: area, centroid, second moments of area, section moduli and radii of gyration, all expressed symbolically so the formulas remain readable.
The collection covers the basics ([rectangle](#rectangle), [circle](#circle), [trapezoid](#trapezoid), [equilateral](#equilateral-triangle) and [right triangles](#right-triangle), [hexagon](#hexagon)), the rolled steel family ([Tee](#tee-section), [Double Tee](#double-tee-section), [angle](#angle-section), [channel](#channel-section), [Zed](#zed-section)), thin-walled pipes ([circular](#circlular-pipe), [elliptical](#elliptical-pipe), [rectangular tube](#rectangular-tube)), and partial-circle shapes ([half](#half-circle), [quarter](#quarter-circle), [sector](#sector), [segment](#segment)).
For irregular outlines the [arbitrary polygon](#polygon) worksheet integrates the same properties from a list of vertex coordinates, while the [solution of a triangle](#solution-of-a-triangle) page provides the trigonometric companion that recovers all six elements from three given ones.
